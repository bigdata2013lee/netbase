内网收集器安装

1. 软\硬件要求

操作系统: Centos 6.3\Centos 6.4\Centos 6.5 64位
CPU: 多核(2核以上)
Mem: 512M 以上
磁盘空间: 40G 以上

2. 安装

解压安装版
1.把innerCollector.tar.gz解压到 /opt目录下
2.通过yum安装SNMP的支持包 yum -y install net-snmp net-snmp-utils
3.创建python的软连接 ln -s /opt/python2.7/bin/python2.7 /opt/netbase4/bin/python


虚拟机安装版
1.要求64位的主机环境
2.使用VMware虚拟机软件打开安装版

3. 配置

配置文件路径: /opt/netbase4/etc/collector.conf
colHost=127.0.0.1 #修改此为内网对应的公网IP
rpcHost=182.254.133.106 #修改为运维通配置服务器IP(182.254.133.106)
colUid 修改为收集器的UID

tcpPort=12368,bootPort=12305,请把这两端口在路由上做映射，使得运维通的服务器(公网IP)，能访问到收集器的这两个端口

4. 运行及停止收集器

1. cd /opt/netbase4/bin
2. ./coll/manager start


#./coll/manager {start|stop|restart|status|list}
收集器管理里程manager，提供 启动|停止|重启|状态|列出所有进程 功能

内网收集器安装完毕后，您需要申请运行证书，请致电运维通客服 4006-3525-00, 向客服人员提供您的登陆帐号，公司\单位名称及收集器MAC地址，确认无误后，将发运行证书到您的邮箱中。

收到运行证书后，请把运行证书(.license.dat文件)上传至收集器设备/opt/netbase4/目录中
